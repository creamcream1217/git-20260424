## Fork & PR 등록 안내
> https://docs.github.com/ko/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo

- 조장님은 본 repo를 `fork` (우측 상단 메뉴) 하세요.
- 개인 계정 하위로 `fork`한 repo에서
   - feature branch에서 `index.html` 파일을 수정하여, 각 조의 github pages 결과 링크를 추가한 변경사항을 커밋을 만듭니다.
   > - **branch name** : `차수th_조장이름` (예- `30th_eunbi-kwon`)
   - 해당 repo로 PR을 보냅니다.
- 제가 확인 후 머지합니다
